Welkom bij onze website!

Wij zijn Merijn van Norden en Nithush Dominicnathan en dit is onze website over Machine Learning. Wij zijn 2 leerlingen van de Sint-Maartnescollege Voorburg. Wij zitten allebei in 4 havo. We hebben beide een E&M vakkenpakket met als vak informatica erin. Informatica is voor ons het enige vak waar je een beetje tot rust kan komen en het werken er ook leuk is. Je krijgt verschillende opdrachten die je allemaal moet doen, maar vaak zijn de opdrachten leuk om te doen. Dat is ook waarom informatica een vak is waarbij je je rust kan houden. We kregen dus nu ook een opdracht. We moesten voor het vak informatica een website maken. We mochten kiezen uit: IT in de zorg, Machine learning, Social bubble en een ander actueel ICT onderwerp. Als je de laatste optie ging kiezen moest je wel toestemming hebben van meneer Logtenberg. Wij kozen allebei voor Machine Learning, omdat Machine Learning voor ons het leukste leek van alle opties. We deden de opdracht dan ook serieus. We probeerden allebei veel designs te bedenken hoe het eruit moest zien en wat we er allemaal in gingen doen. Uiteindelijk ziet onze website er dus zo uit. Hopelijk vinden jullie onze website goed eruit. Jullie kunnen onze website ook volgen op Instagram, Facebook en Twitter. Ze staan er allemaal hieronder. Wil je up to date blijven van onze website? Dan kan je het beste ons volgen op alle 3 de social media platforms.

Merijn van Norden
