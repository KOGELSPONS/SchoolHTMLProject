  1. Machine Learning:

    2. Machine learning het klinkt als een pre-apocalyptisch term. Over dat er robots mensen hun banen overnemen en te veel weten over hun eigenaren. En hun zwakke plekken gebruiken om hun eigenaren naar te halen, maar in realiteit is dit niet zo. Machine learning is namelijk een soort algoritme. Een algoritme in het kort is een verzameling van instructie om een wiskundige of informatief probleem op te lossen. Machine learning valt dus ook onder de algoritme het lost namelijk een vraag op en geeft een duidelijk antwoord terug. Hieronder zullen verschillende doeleinde van een Algoritme uitgelegd.

  2. Gezichtsherkenning:

    3. Je geeft het Algoritme en foto van een persoon. En vergelijkt die met foto’s uit een database een locatie waar je data kan opslaan. Een zal je een naam terug geven. Deze algoritme wordt getraind door aan te geven waar het naar moet kijken. Dus bepaalde aspecten van het gezicht ogen, vorm, huidskleur, haar. En dan wordt het algoritme losgelaten op foto’s van personen. Dit algoritme vergelijkt dan het verkregen antwoordt met het eerder opgegeven antwoordt. En gebruikt deze data zichzelf te trainen.

  3. Captcha:

    4. Dit wordt door veel websites gebruikt, waarschijnlijk heb jij dit ook wel eens gedaan. Een test om te kijken of je geen robot bent. Dat is dus een Captcha text. En wordt gebruikt om een Machine Learning Algoritme te trainen. Jouw antwoorden worden gebruikt om het algoritme die kijk of jij geen robot bent te trainen. Dit wordt gedaan door meerdere mensen een nieuw plaatje te geven en te kijken want hun antwoord is en als dan 90% het een bus vond, maar het algoritme dacht dat het een vrachtwagen was. Dan wordt dat antwoordt gebruikt om het algoritme te trainen.

  4. Soorten machine learning:

    5. Er worden 2 soorten algoritmen gebruikt voor machine learning:

    • Supervised machine learning:

      • Bij supervised machine learning leert een datawetenschapper wat het algoritme moet doen of wanneer het algoritme een conclusie moet trekken. Zoals wij al van jongs af aan iets geleerd krijgen zoals dat dit een vierkant is en dat het daar bij past of wel verschillende soorten groenten leren. Algoritmen leren dit ook door data wetenschappers die het algoritme het leren.

    • Unsupervised machine learning:

      • Bij supervised machine learning biedt de mens heel veel begeleiding aan aan de computer, maar bij unsupervised machine learning leert de computer zelf om complexe processen en patronen te identificeren. Dus bij een supervised machine learning biedt de mens geen begeleiding aan aan de computer. Bij unsupervised machine learning traint de computer zichzelf op basis van data. Die data bevat geen labels of specifieke dingen die een gedefinieerde output kan hebben.
